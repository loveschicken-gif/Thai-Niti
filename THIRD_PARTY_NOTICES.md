# Third-Party Notices (Data Sources)

Thai statutory text remains the source of truth for gameplay and legal recall use cases.

## Derived/Processed Data in This Repository

The following files are generated by project scripts and contain transformed third-party data:

- `data/processed/thai_niti_normalized.json`
- `data/processed/thai_niti_deduped.json`
- `data/processed/dataset_profile_report.json`
- `data/processed/open_law_normalized.json`
- `data/processed/open_law_profile_report.json`
- `data/processed/statute_tracks/*.json`
- `data/eval/*` (evaluation details, summaries, CSV tables)

## Script-to-Source Mapping

- `scripts/prepare_thai_niti_data.py` -> `airesearch/WangchanX-Legal-ThaiCCL-RAG`
- `scripts/prepare_open_law_corpora.py` -> `open-law-data-thailand/soc-ratchakitcha`, `open-law-data-thailand/ocs-krisdika`
- `scripts/ingest_statute_tracks.py` -> PyThaiNLP criminal CSV release, `pythainlp/thailaw-v1.0`, `iapp/rag_thai_laws`
- `scripts/evaluate_retrieval.py` and `scripts/tune_reranker.py` -> `airesearch/WangchanX-Legal-ThaiCCL-RAG`
